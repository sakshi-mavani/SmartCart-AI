## Key Engineering Achievements

* Built an end-to-end machine learning recommendation platform from raw Amazon review data to cloud deployment.
* Processed and engineered features from over 701K real-world e-commerce reviews.
* Developed a TF-IDF + Cosine Similarity recommendation engine for product discovery.
* Implemented cold-start handling using popularity-based fallback recommendations.
* Trained and compared multiple machine learning models including Logistic Regression, Random Forest, and XGBoost.
* Exposed recommendation and prediction models through production-grade FastAPI REST APIs.
* Containerized the application using Docker for reproducible deployment.
* Deployed the complete system on Render Cloud.
* Tracked machine learning experiments using MLflow.
* Built an interactive Streamlit analytics dashboard for business insights and visual exploration.
